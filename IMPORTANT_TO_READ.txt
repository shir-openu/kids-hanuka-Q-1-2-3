================================================================================

PROBLEM:
--------
When Claude Code tried to CREATE geometric figures from scratch using SVG code,
the results were NOT ACCURATE. The angles, proportions, and positions were
approximations that DID NOT MATCH the original homework drawings.

THIS IS CRITICAL FOR KIDS because:
- Misleading diagrams confuse children
- Wrong visual representation leads to wrong understanding
- Geometry requires EXACT figures to learn correctly

WHAT WENT WRONG (v1, v2):
-------------------------
Claude attempted to manually write SVG coordinates like:

    <line x1="400" y1="45" x2="40" y2="200"/>  <!-- guessed coordinates -->
    <path d="M 400 45 L 360 85 A 45 45 0 0 0 375 90 Z"/>  <!-- approximated arc -->

Problems with this approach:
1. Claude cannot "see" geometry - only generates code
2. Angle calculations were mathematical approximations
3. Triangle proportions didn't match the original PDF
4. Kids would see DIFFERENT figures than in their homework!

================================================================================
‚úÖ  THE SOLUTION - USE EXACT SVG FILES (HIGH IMPORTANCE!)
================================================================================

SOLUTION:
---------
Instead of Claude creating figures, we used the EXACT SVG files exported from
the original PDF:

    q1_drawing.svg  (exact figure for question 1)
    q2_drawing.svg  (exact figure for question 2)
    q3_drawing.svg  (exact figure for question 3)

These SVG files contain the PRECISE geometry from the teacher's homework.

HOW TO EMBED EXACT SVG FILES IN HTML:
-------------------------------------
WRONG (v1, v2 - manual SVG code):

    <svg width="450" height="280">
      <line x1="400" y1="45" x2="40" y2="200"/>  <!-- INACCURATE! -->
      ...manually written coordinates...
    </svg>

CORRECT (v4 - reference external SVG file):

    <img src="q1_drawing.svg" alt="◊©◊®◊ò◊ï◊ò ◊©◊ê◊ú◊î 1">

This way, the HTML simply DISPLAYS the exact original figure without
any modification or approximation.
